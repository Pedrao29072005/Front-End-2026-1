# Programação Front End - 3º Semestre UniCesumar

Repositório contendo projetos, exercícios e avaliações desenvolvidos na disciplina de **Programação Front End** do 3º semestre do curso de Análise e Desenvolvimento de Sistemas da UniCesumar.

## 📋 Descrição

Este repositório organiza todos os trabalhos práticos e conceituais de desenvolvimento front-end, incluindo projetos de JavaScript, HTML, CSS, além de provas e atividades complementares realizadas ao longo do semestre.

## 📚 Projetos e Entregas

### Trabalhos Principais (trabalho1.html até trabalho5.html)
Cinco trabalhos práticos que cobrem diferentes conceitos e tecnologias de front-end durante o semestre.

### Exercícios de Revisão (revisão.html até revisão4.html)
Quatro conjuntos de exercícios para consolidação dos conhecimentos adquiridos.

### Projetos Especiais

| Projeto | Descrição |
|---------|-----------|
| **Clicker** | Jogo interativo tipo clicker com contadores e eventos JavaScript |
| **Lâmpada** | Projeto interativo que simula acender/apagar uma lâmpada com imagens |
| **Pomodoro** | Aplicação de técnica Pomodoro com timer e controle de tempo |
| **Quadrado** | Jogo ou exercício envolvendo quadrados e interatividade |
| **Bichinho** | Projeto especial com animações e interações |

### Avaliação (Prova/)
Pasta contendo arquivos da prova semestral com múltiplas questões e validações JavaScript.

## 🛠 Tecnologias Utilizadas

- **HTML5** - Estrutura e marcação semântica
- **CSS3** - Estilização e design responsivo
- **JavaScript (ES6+)** - Interatividade e lógica de programação
- **Imagens (PNG)** - Recursos visuais dos projetos

## 💻 Como Utilizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Pedrao29072005/Front-End-2026-1.git
   ```

2. Abra qualquer arquivo `.html` em seu navegador preferido

3. Navegue pelos diferentes projetos nas pastas `Entregas/`
